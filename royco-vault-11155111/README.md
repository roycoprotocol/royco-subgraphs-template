## Subgraph Commands

### Deploy

```bash
goldsky subgraph deploy royco-vault-11155111/0.0.22 --path .
```

### Pause

```bash
goldsky subgraph pause royco-vault-11155111/0.0.22
```

### Delete

```bash
goldsky subgraph delete royco-vault-11155111/0.0.22
```

### Stop

goldsky pipeline stop royco-vault-11155111

### Delete
