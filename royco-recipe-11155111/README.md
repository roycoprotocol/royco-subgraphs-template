## Subgraph Commands

### Deploy

```bash
goldsky subgraph deploy royco-recipe-11155111/0.0.21 --path .
```

### Pause

```bash
goldsky subgraph pause royco-recipe-11155111/0.0.21
```

### Delete

```bash
goldsky subgraph delete royco-recipe-11155111/0.0.21
```
