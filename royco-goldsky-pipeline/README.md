## Pipeline Commands

### Install

```bash
npm install
```

### Bash Script

```bash
./deploy-new-pipelines.sh
```

### Prepare

```bash
npm run prepare
```

### Update

```bash
goldsky pipeline apply royco-goldksy-pipeline.yaml
```

### Stop

```bash
goldsky pipeline stop royco-goldksy-pipeline
```

### Delete

```bash
goldsky pipeline delete royco-goldksy-pipeline
```
